VIM
===


Opening Files
-------------
* ``vim``
* ``vim myfile.txt``
* ``vim /tmp/myfile.txt``
* ``vim /tmp/myfile.txt +13``


Edit/Command Mode
-----------------
* ``a``
* ``i``
* ``esc``

Writing and Quit
----------------
* ``:w`` - write
* ``:q`` - quit
* ``:q!`` - quit without saving
* ``:wq`` - write and quit
